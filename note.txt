HOF (High Order Function) adalah function yang menerima function sebagai argument/parameter dan dapat di return lagi sebagai function biasa. HOF digunakan agar code kita jadi lebih mudah dibaca dan dimengerti dengan meringkasnya.

HOF terdiri dar beberapa jenis yang disediakan oleh javascript yakni:
    1. .forEach()
    2. .filter()
    3. .reduce()
    4. .map()

selain keempat jenis HOF yang disediakan oleh javascript, kita bisa custom sendiri juga HOF ini.
